***Запуск websocket API сервера***

З директорії *src/server*:

Для встановлення залежностей
- *npm install*
 
Для запуску проекту
- *npm start*

Для тестів
- *npm test*


***Запуск сервера статіки:***

З директорії *src/client*:

Для встановлення залежностей
- *npm install*

Для запуску проекту
- *npm start*

В браузері: 
- *localhost:8080*
